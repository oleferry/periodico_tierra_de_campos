# Checklist de publicación
## El Terracampino

### Hechos

- [ ] Todas las cifras tienen fuente y fecha.
- [ ] Los nombres están correctamente escritos.
- [ ] Las citas proceden de entrevista o documento verificable.
- [ ] No hay afirmaciones causales no demostradas.
- [ ] Las inferencias están identificadas.

### Storytelling

- [ ] La apertura contiene una escena concreta.
- [ ] La naturaleza de la escena está explicada.
- [ ] No se presenta ficción como hecho.
- [ ] La historia sirve al tema.
- [ ] El cierre vuelve a la apertura.

### Tierra de Campos

- [ ] Se nombran lugares concretos.
- [ ] La aplicación local está desarrollada.
- [ ] Se explica qué podría funcionar y qué no.
- [ ] Hay al menos una posible voz local.

### Estilo

- [ ] Los párrafos desarrollan una idea completa y tienen, por lo general, 3-6 frases.
- [ ] El artículo tiene aproximadamente 14-22 párrafos de cuerpo.
- [ ] No hay más de 3 párrafos de una sola frase.
- [ ] Los subtítulos aparecen cada 300-450 palabras aproximadamente.
- [ ] Los saltos de párrafo responden a cambios de escena o argumento, no a cada conjunción.
- [ ] Sin humor intencional.
- [ ] Sin emojis ni hashtags.
- [ ] Sin lenguaje de nota de prensa.
- [ ] Sin clichés de IA.
- [ ] No hay una sucesión de frases aisladas ni una conclusión escolar.

### Riesgo

- [ ] Se revisaron datos personales.
- [ ] Se protegió a menores.
- [ ] Se revisaron acusaciones y conflictos.
- [ ] Se ofrece fuente original.
- [ ] La pieza no parece comunicación oficial.

### Distribución

- [ ] Titular web.
- [ ] Entradilla.
- [ ] Metadescripción.
- [ ] Newsletter.
- [ ] Telegram.
- [ ] Pieza social.
- [ ] Imagen con derechos claros.
