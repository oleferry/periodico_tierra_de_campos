# Sistema de marca de El Terracampino - versión 1.2

Este paquete reúne las reglas que deben gobernar la marca, la redacción, el storytelling, el diseño y la automatización editorial.

## Orden de uso

1. `01_marca/brand_content_book.md`
2. `02_editorial/guia_storytelling_articulos.md`
3. `02_editorial/checklist_publicacion.md`
4. `03_visual/kit_marca_visual.md`
5. `04_automatizacion/prompt_maestro_articulos.md`
6. `05_fuentes/guia_estilo_original.md`

## Regla operativa

Antes de producir un artículo largo:

1. preparar dossier;
2. completar ficha previa;
3. redactar borrador factual;
4. aplicar storytelling;
5. pasar checklist;
6. generar adaptaciones;
7. revisar manualmente si hay riesgo medio o alto.

## Alcance

Este sistema se aplica a:

- artículos de investigación;
- newsletters;
- resúmenes municipales;
- deportes;
- tiempo;
- campo y huerta;
- agenda;
- historias locales;
- negocios;
- redes sociales.

## Prioridad

Cuando exista conflicto:

1. precisión;
2. transparencia;
3. utilidad;
4. claridad;
5. estilo.

El estilo nunca debe ganar a los hechos.

## Cambio editorial de la versión 1.2

La versión 1.2 corrige la cadencia de los artículos largos. La guía de estilo original sigue siendo una referencia para voz, claridad y ausencia de clichés, pero sus reglas de párrafo ultracorto no se aplican literalmente a reportajes de 6-8 minutos.

En artículos, reportajes y ensayos de newsletter mandan estas reglas:

- párrafos narrativos de 3-6 frases;
- entre 45 y 90 palabras por párrafo como rango habitual;
- entre 14 y 22 párrafos de cuerpo para una pieza de 1.100-1.450 palabras;
- máximo de 3 párrafos de una sola frase en todo el artículo;
- ningún párrafo de una sola palabra como obligación;
- no se rompe párrafo automáticamente antes de «y», «pero», «porque» u «o»;
- los subtítulos aparecen cada 300-450 palabras, no para trocear cada idea;
- el ritmo se construye dentro del párrafo mediante longitud de frase, puntuación y transiciones.

Las piezas breves, alertas, Telegram y redes sí pueden usar una fragmentación mayor.
